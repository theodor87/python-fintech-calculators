# Python FinTech Calculators

A collection of simple Python scripts for basic forex and crypto trading calculations.

## Included calculators
- Forex Lot Size Calculator
- Forex Profit Calculator
- Crypto Profit Calculator
- Forex Spread Cost Calculator
- Simple Password Validator

## Purpose
These scripts were created as practice projects while learning Python fundamentals, focusing on real-world financial and trading use cases.

## Technologies
